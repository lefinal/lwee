# Stream

`stream`

Streams allow easy usage of LWEE SDKs and support handling multiple streams at once.
For more information regarding streams, refer to the [streams section](./../streams.mdx).

## Configuration

```yaml
source: ""
provideAs: stream
# Name of the input stream that data will be read from.
streamName: "entries"
```

::::note
Make sure to fully read streams.
If you do not need the data, discard it or remove the input from the action.
::::
