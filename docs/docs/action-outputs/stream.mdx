# Stream

`stream`

Streams allow easy usage of LWEE SDKs and support handling multiple streams at once.
For more information regarding streams, refer to the [streams section](./../streams.mdx).

## Configuration

```yaml
providedAs: stream
# Name of the output stream that data will be read from.
streamName: "entries"
```

::::note
Make sure to close streams after you have transmitted all data.
::::
